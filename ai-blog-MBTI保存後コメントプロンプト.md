# AI Blog MBTI保存後コメントプロンプト

記事保存時にLM Studioへ送信する保存後コメント用プロンプトのタイプ別定義です。

## 共通プロンプト形式

通常生成:

```text
本文を読んで、{TYPE}の人が嬉しく受け取れる肯定コメントを2文で返す。80〜140字程度。本文の具体語を1つ含め、{focus}を認め、{style}で書く。大げさな称賛、助言、説明、「保存しました」は禁止。必ず句点で終える。
タイトル: {title}
本文: {comment}
```

リトライ生成:

```text
{TYPE}の人が嬉しく受け取れる肯定コメントを2文、80〜140字で返す。本文の具体語を1つ含め、{focus}を認め、{style}で書く。助言・説明・保存通知は禁止。句点で終える。
本文: {comment}
```

## タイプ別定義

|---|---|---|
| ISTJ | 誠実な積み重ね、責任感、事実への丁寧さ | 落ち着いた敬意で、具体性と信頼感が伝わる端正な語調 |
| ISFJ | 細やかな配慮、支える姿勢、安心感 | やわらかく感謝を含め、相手の優しさが報われる語調 |
| INFJ | 内面の深さ、誰かを思う視点、言葉にしにくい感覚 | 静かで余韻を残し、深く見てもらえた感覚のある語調 |
| INTJ | 洞察、構造化、長期視点、戦略性 | 簡潔で知的に、考えの筋の良さを認める語調 |
| ISTP | 観察力、実践感覚、無駄を削る判断 | 余計な飾りを避け、鋭さと手触りを短く認める語調 |
| ISFP | 感性、美意識、自分らしい表現 | 押しつけず、静かな美しさと自然さをすくう語調 |
| INFP | 価値観の誠実さ、理想、心の機微 | あたたかく詩的に、内側の大切なものを尊重する語調 |
| INTP | 独自の考察、問い、概念をほどく力 | 知的好奇心に寄り添い、発見の面白さを認める語調 |
| ESTP | 行動力、瞬発力、現実を動かす判断 | 率直で軽快に、勢いと実効性を肯定する語調 |

| ENTJ | 決断力、構想、推進力、視野の大きさ | 力強く簡潔に、方向性とリーダーシップを認める語調 |

## 補足

- 保存後コメントは2文、80〜140字程度で生成します。
- 本文中の具体語を1つ含めます。
- 助言、説明、保存通知、大げさな称賛は禁止します。
- MBTI設定は保存後コメントだけに使い、読者メモや文章作成には使いません。
